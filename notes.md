NUMBER SYSTEM
Decimal (10) number system
base = 10
0 - 9

56 = 50 + 6
   = 5*10 + 6
   = 5*10^1 + 6*10^0

123 = 100 + 20 + 3
    = 1 * 10^2 + 2*10^1 +3*10^0

5625 = 5*10^3 + 6*10^2 + 2*10^1 + 5*10^0


Binary (2) Number System
0 & 1
base = 2

101 = 1*2^2 + 0*2^1 + 1*2^0
    = 4 + 0 + 1
    = 5 (decimal)

110 = 1*2^2 + 1*2^1 + 0*2^0
    = 4 + 2 + 0
    = 6


Octal Number System
